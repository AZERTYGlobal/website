Ce guide résume les principales fonctionnalités d'AZERTY Global.

────────────────────────────────────────────────────────────
🔄  LES 5 CHANGEMENTS PAR RAPPORT À L'AZERTY TRADITIONNEL DE WINDOWS
────────────────────────────────────────────────────────────

1. VERROUILLAGE MAJUSCULE INTELLIGENT
   → N'affecte QUE les lettres
   → Permet de taper É, È, Ç, À en un appui

2. POINT EN ACCÈS DIRECT
   → Le point (.) est maintenant en accès direct
   → Le point-virgule (;) passe en Maj + .

3. @ ET # SUR LA TOUCHE ² (en haut à gauche)
   → @ en accès direct, # en Maj

4. SYMBOLES DE PROGRAMMATION SUR LA RANGÉE DE REPOS
   → AltGr + D/F → { }    AltGr + J/K → [ ]
   → AltGr + G → \        AltGr + H → |
   → AltGr + N → ~        AltGr + Maj + N → ` (backtick)

5. ACCENTS INTERNATIONAUX SUR LA TOUCHE Ù
   → Accent aigu (´) et grave (`) remplacent le ù et le %
   → Le ù passe en AltGr + U, le % sur Maj + )


────────────────────────────────────────────────────────────
🔠  MAJUSCULES ACCENTUÉES
────────────────────────────────────────────────────────────

Activez Verr. Maj. puis tapez la lettre :
  é → É    è → È    ç → Ç    à → À


────────────────────────────────────────────────────────────
✒️  TYPOGRAPHIE FRANÇAISE
────────────────────────────────────────────────────────────

  • AltGr + O/A          → œ æ (et Œ Æ avec Maj)
  • AltGr + W/X          → « »
  • AltGr + T            → – (tiret demi-cadratin)
  • AltGr + Maj + T      → — (tiret cadratin)
  • AltGr + 6            → ‑ (tiret insécable)
  • AltGr + 4            → ' (apostrophe typographique)
  • Maj + ! puis Point   → … (points de suspension)
  • AltGr + Maj + Point  → · (point médian)
  • AltGr + Espace       → espace insécable
  • AltGr + Maj + Espace → espace fine insécable


────────────────────────────────────────────────────────────
🌍  LANGUES EUROPÉENNES
────────────────────────────────────────────────────────────

ESPAGNOL :
  • Circonflexe puis N → ñ   • AltGr + ! → ¡   • AltGr + Maj + virgule → ¿

ALLEMAND :
  • AltGr + B → ß   • Maj + Circonflexe puis A/O/U → ä ö ü

PORTUGAIS :
  • AltGr + Accent aigu puis A/O → ã õ

ITALIEN :
  • Maj + Accent aigu puis A/E/I/O/U → à è ì ò ù


────────────────────────────────────────────────────────────
⚡  SYMBOLES UTILES
────────────────────────────────────────────────────────────

TOUCHE MORTE SCIENTIFIQUE (AltGr + =) :
  • puis Espace → ±    • puis 8 → ∞      • puis V → √
  • puis / → ÷         • puis * → ×      • puis = → ≠

TOUCHE MORTE SYMBOLES DIVERS (AltGr + @) :
  • puis 1/2/3 = ¼ ½ ¾     (fractions)
  • puis Q/D/F = ← → ↔     (flèches horizontales)
  • puis W     = ↗ ↘       (flèches diagonales)
  • puis V/X   = ✅ ❌     (coche / croix)
