🔠 Majuscules accentuées

Comment taper les majuscules accentuées (É, È, À, Ç) ?

Avec AZERTY Global, c'est enfin simple !

  • Activez Verr. Maj puis tapez la lettre accentuée (é → É, è → È, à → À, ç → Ç)
  • Ou utilisez ^ puis la lettre accentuée

Le Verr. Maj intelligent met automatiquement les accents en majuscule.

Méthode classique : codes Alt interminables (Alt+144, Alt+0192, etc.) ou copier-coller

✒️ Typographie française

Comment taper la ponctuation française (« », —, espaces insécables, …) ?

AZERTY Global rend la typographie française professionnelle accessible :

Les guillemets français « »

  • AltGr + W → «
  • AltGr + X → »

W et X sont côte à côte sur la rangée du bas, facile à retenir !

Les tirets

  • AltGr + T → – (tiret demi-cadratin, pour les intervalles : 2020–2025)
  • AltGr + Maj + T → — (tiret cadratin, pour les dialogues)
  • AltGr + - → ‑ (tiret insécable)

Les espaces insécables

  • AltGr + Espace → espace insécable (avant : ; ! ?)
  • AltGr + Maj + Espace → espace fine insécable

L'espace insécable empêche les retours à la ligne indésirables avant la ponctuation.

Les points de suspension …

  • Maj + ! (touche morte ponctuation) puis . → …

L’apostrophe typographique ‘

  • AltGr + 4 (la touche de l’apostrophe) → ’

L'apostrophe typographique (’) est plus élégante que l'apostrophe droite (').

Les lettres spéciales (œ, æ)

  • AltGr + O → œ (cœur, œuf)
  • AltGr + A → æ (Lætitia, ex æquo)

Résumé : Avec AZERTY Global, vous pouvez écrire « Bonjour ! » — avec une typographie professionnelle…

⌨️ Symboles

Symboles dev : @ # { } [ ] \ | ~ ` — où sont-ils ?

Avec AZERTY Global, les symboles de programmation sont plus accessibles :

  • @ : touche en haut à gauche (ancienne ²)
  • # : Maj + @
  • ~ : AltGr + N (tilde vif)
  • ` : Maj + AltGr + N (backtick vif)

Sur la rangée de repos avec AltGr :

  • { } : AltGr + D / F
  • [ ] : AltGr + J / K
  • \ : AltGr + G
  • | : AltGr + H

→ Voir tous les symboles pour développeurs

Symboles courants : →, ×, ©, •, ½, ≠ — comment les taper ?

AZERTY Global donne accès à des centaines de symboles via les touches mortes :

Accès direct

  • ≤ : AltGr + <
  • ≥ : Maj + AltGr + >

Touche morte scientifique (AltGr + =)

  • ± : puis Espace
  • ≠ : puis =
  • × : puis X

Touche morte symboles (AltGr + @)

  • → : puis D (ou Espace)
  • © : puis C
  • ½ : puis 2

Touche morte ponctuation (Maj + !)

  • • (puce) : puis .

❓ Questions générales

C'est quoi AZERTY Global ?

AZERTY Global est une disposition de clavier française améliorée qui corrige les défauts de l'AZERTY traditionnel :

  • Majuscules accentuées (É, À, Ç) accessibles directement
  • Guillemets français « » faciles
  • Caractères spéciaux (œ, æ, …) intégrés
  • Verr. Maj intelligent

C'est une évolution, pas une révolution : 5 changements bénéfiques et nécessaires seulement.

→ Apprendre les 5 changements en 2 minutes · Voir le comparatif détaillé

Combien de temps pour s'adapter ?

La plupart des utilisateurs s'adaptent en quelques minutes à quelques heures. Il n'y a que 5 changements majeurs à mémoriser.

Moins de 1% de vos frappes seront différentes ! Les lettres et chiffres ne bougent pas.

Dois-je acheter un nouveau clavier ?

Non, absolument pas !

AZERTY Global est une solution purement logicielle. Elle s'installe sur votre Windows et redéfinit ce que font vos touches actuelles. Votre matériel existant est 100% compatible.

Est-ce compatible avec les jeux vidéo (ZQSD) ?

OUI ABSOLUMENT.

Les touches physiques ne bougent pas. Vos touches de déplacement ZQSD restent exactement au même endroit.

Les jeux reconnaissent le clavier comme un AZERTY standard. Aucune reconfiguration nécessaire.

Comment garder AZERTY Global par défaut après un redémarrage ?

Application Microsoft Store

Clic droit sur l'icône AG dans la barre des tâches → Lancer au démarrage de Windows.

Installateur EXE

Si Windows revient à votre ancien clavier après un redémarrage :

Option 1 (recommandé) : Supprimez les anciens claviers dont vous n'avez plus besoin.
Paramètres → Heure et langue → Langue et région → Français (⋯) → Options de langue → Claviers → Supprimer les claviers inutiles

Option 2 : Si vous devez garder plusieurs claviers :
Paramètres → Heure et langue → Saisie → Paramètres de clavier avancés → « Remplacer la méthode d'entrée par défaut » → choisissez AZERTY Global 2026

macOS

Une fois ajouté dans Réglages Système → Clavier → Entrée de texte, AZERTY Global reste disponible après redémarrage. Supprimez les claviers inutiles pour qu'il soit le seul actif.

Linux

Le script d'installation configure AZERTY Global de façon permanente. Il reste actif après redémarrage.

Disponible sur Mac et Linux ?

Oui ! AZERTY Global est disponible sur Windows (7, 8, 10, 11), macOS (10.12+) et Linux (Ubuntu, Debian, Fedora, Arch...).

Comment désinstaller AZERTY Global ?

C'est très simple et sans risque. AZERTY Global ne laisse aucune trace après désinstallation.

Application Microsoft Store

Win + I → Applications → Applications installées → Cherchez « AZERTY Global » → Désinstaller.

Installateur EXE

  1. Retirez le clavier : Paramètres → Heure et langue → Langue et région → Français (⋯) → Options de langue → Sélectionnez AZERTY Global → Supprimer.
  2. Relancez l'installeur EXE pour désinstaller complètement.

macOS

  1. Retirez le clavier dans Réglages Système → Clavier → Entrée de texte.
  2. Supprimez le fichier AZERTY Global (Beta).keylayout de ~/Library/Keyboard Layouts/.
  3. Redémarrez votre session.

Linux

  1. Retirez AZERTY Global de vos sources de saisie.
  2. Supprimez le fichier : sudo rm /usr/share/X11/xkb/symbols/azerty_global
  3. Restaurez votre ancien ~/.XCompose si besoin (un backup a été créé).
  4. Redémarrez votre session.
AZERTY Global vs AFNOR : lequel choisir ?

Les deux projets corrigent l'AZERTY traditionnel, mais avec des approches très différentes :

  • AZERTY Global : 5 changements seulement, transition douce, Verr. Maj intelligent pour les majuscules accentuées, symboles dev sur la rangée de repos. Gratuit, open source, disponible sur le Microsoft Store.
  • AFNOR (NF Z71-300) : refonte complète avec 60+ caractères repositionnés, nécessite un apprentissage plus long. Utilise AltGr + Maj pour les majuscules accentuées.

En résumé : AZERTY Global privilégie l'adoption rapide (quelques minutes), l'AFNOR vise l'exhaustivité au prix d'un réapprentissage plus conséquent.
