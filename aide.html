🔠 Majuscules accentuées

Comment faire un É (E accent aigu majuscule) ?

Avec AZERTY Global :

  • Verr. Maj puis é → É
  • ^ (circonflexe) puis é → É
  • ´ (accent aigu) puis Maj + E → É

Le Verr. Maj intelligent met les lettres accentuées en majuscule automatiquement.

Méthode classique : les fameux Alt + 144, Alt + 0201 (ou Alt + 45456 😉)

Astuce Pro : Mettre un accent sur n'importe quoi (Ŵ, 4́, M̀...)

AZERTY Global permet d'utiliser les diacritiques combinants pour accentuer des lettres ou symboles qui ne le sont pas habituellement.

La règle est inversée :

  1. Tapez d'abord votre caractère (ex: W)
  2. Tapez la touche morte de l'accent DEUX FOIS (ex: ^ puis ^)

Résultat : Ŵ. Marche aussi pour les accents aigus (Maj + é x2), graves, etc.

Développeurs : Où sont le backslash \ et le pipe | ?

Ils sont sur la rangée de repos (home row) pour un accès ergonomique :

  • Backslash (\) : AltGr + G
  • Barre verticale / Pipe (|) : AltGr + H

Position idéale sans quitter la position dactylo !

Où est le u accent grave ù ?

Trois méthodes, de la plus rapide à la plus lente :

  1. ´ puis V (accent aigu + V) — la plus rapide !
  2. Alt Gr + U — accès direct
  3. ` puis U (accent grave + U) — via touche morte

Pourquoi ce déplacement ? La touche historique (ù / %) a été repensée pour accueillir les accents aigus et graves en touches mortes. Cela permet d'accentuer facilement n'importe quelle lettre (á, ó, ǹ...).

Où est le pourcent % ?

Il est sur Maj + ) (touche à droite du 0).

Pourquoi ce déplacement ? Comme pour le ù, l'ancienne touche a été réattribuée pour placer les accents aigus et graves en accès direct (touches mortes). Le % est désormais deux touches au-dessus.

❓ Questions générales

C'est quoi le clavier AZERTY Global ?

AZERTY Global est une disposition de clavier française améliorée qui corrige les défauts de l'AZERTY traditionnel :

  • Majuscules accentuées (É, À, Ç) accessibles directement
  • Guillemets français « » faciles
  • Caractères spéciaux (œ, æ, …) intégrés
  • Verr. Maj intelligent

C'est une évolution, pas une révolution : 5 changements majeurs seulement.

Différence entre AZERTY et AZERTY AFNOR ?

L'AFNOR (norme NF Z71-300 de 2019) propose une refonte complète du clavier, avec de nombreuses touches déplacées.

AZERTY Global prend l'approche inverse : on garde l'AZERTY Windows et on corrige uniquement les défauts. La transition est beaucoup plus douce.

Voir le comparatif détaillé →

Comment changer disposition clavier Windows ?

Pour installer AZERTY Global :

  1. Téléchargez l'installateur
  2. Lancez l'installation (droits admin requis)
  3. Sélectionnez AZERTY Global dans les paramètres Windows

Basculez entre claviers avec Win + Espace.

Comment désinstaller AZERTY Global ?

C'est très simple et sans risque :

  1. Allez dans les Paramètres Windows > Heure et langue > Langue.
  2. Cliquez sur votre langue (Français) > Options.
  3. Dans "Claviers", sélectionnez AZERTY Global et cliquez sur Supprimer.

Vous revenez instantanément à votre clavier précédent.

Est-ce que mes raccourcis (Ctrl+C, Ctrl+V) changent ?

Non ! Tous vos raccourcis habituels restent identiques. Seuls les caractères produits par certaines touches changent, pas les raccourcis système.

Est-ce compatible avec les jeux vidéo (ZQSD) ?

OUI ABSOLUMENT.

Les touches physiques ne bougent pas. Vos touches de déplacement ZQSD restent exactement au même endroit.

Les jeux reconnaissent le clavier comme un AZERTY standard. Aucune reconfiguration nécessaire.

Disponible sur Mac et Linux ?

Oui ! AZERTY Global est disponible sur Windows (7, 8, 10, 11), macOS (10.12+) et Linux (Ubuntu, Debian, Fedora, Arch...).

Télécharger pour votre système →

Combien de temps pour s'adapter ?

La plupart des utilisateurs s'adaptent en quelques minutes à quelques heures. Il n'y a que 5 changements majeurs à mémoriser :

  • Point en accès direct (. en base, ; en Maj)
  • Guillemets sur AltGr + W/X
  • Crochets/accolades sur la rangée de repos
  • Verr. Maj intelligent
  • Touches mortes pour accents
Dois-je acheter un nouveau clavier ?

Non, absolument pas !

AZERTY Global est une solution purement logicielle. Elle s'installe sur votre Windows et redéfinit ce que font vos touches actuelles. Votre matériel existant est 100% compatible.

Ai-je besoin d’autocollants (stickers) ?

Pas obligatoirement.

Comme la grande majorité des touches (lettres A-Z, chiffres) ne bougent pas, vous n'avez pas besoin de repères visuels pour tout.

Pour les quelques nouveaux caractères (guillemets, majuscules accentuées...), la mémoire musculaire s'installe très vite. Des stickers peuvent aider au début pour l'apprentissage, mais ne sont pas indispensables.

Où acheter un clavier AZERTY Global ?

Il n'existe pas encore de clavier physique dédié.

Pour l'instant, aucun constructeur ne commercialise de clavier avec le marquage AZERTY Global.

Mais bonne nouvelle : vous n'en avez pas besoin ! Le logiciel fonctionne sur n'importe quel clavier AZERTY standard. C'est tout l'intérêt du projet : moderniser votre matériel existant gratuitement.

🌍 Taper en langues européennes

Comment taper en espagnol (ñ, ¿, ¡, accents) ?

AZERTY Global supporte nativement tous les caractères espagnols :

Le ñ (eñe)

  • ^ (circonflexe) puis N → ñ
  • ^ puis Maj + N → Ñ
  • Alternative : AltGr + ù (touche morte tilde) puis N

Les points d'interrogation et d'exclamation inversés

  • AltGr + Maj + , → ¿
  • AltGr + ! → ¡

Les voyelles accentuées (á, é, í, ó, ú)

Utilisez la touche morte accent aigu :

  • ´ (accent aigu) puis la voyelle → á, é, í, ó, ú
  • ´ puis Maj + voyelle → Á, É, Í, Ó, Ú

La touche accent aigu (´) est sur la même touche que l'ancien ù, accessible directement.

Le ü (diérèse)

  • Maj + ^ (tréma) puis U → ü
  • Maj + ^ puis Maj + U → Ü

Résumé : Avec AZERTY Global, vous pouvez écrire "¿Cómo estás?" ou "España" sans effort !

Comment taper en allemand (ä, ö, ü, ß) ?

AZERTY Global supporte nativement tous les caractères allemands :

Les Umlauts (ä, ö, ü)

Utilisez la touche morte tréma :

  • Maj + ^ (tréma) puis A → ä
  • Maj + ^ puis O → ö
  • Maj + ^ puis U → ü

Pour les majuscules : Maj + ^ puis Maj + voyelle → Ä, Ö, Ü

Le Eszett (ß)

  • AltGr + B → ß
  • AltGr + Maj + B → ẞ (majuscule)

Résumé : Avec AZERTY Global, vous pouvez écrire "Größe" ou "Süß" facilement !

Comment taper en portugais (ã, õ, ç, accents) ?

AZERTY Global supporte nativement tous les caractères portugais :

Le tilde (ã, õ)

  • ^ (circonflexe) puis P → ã (raccourci)
  • AltGr + ù (touche morte tilde) puis A → ã
  • AltGr + ù puis O → õ

Pour les majuscules : utilisez Maj + la lettre après la touche morte.

Le ç (cédille)

  • ç est directement sur le clavier (touche 9)
  • Verr. Maj puis ç → Ç

Les accents (á, é, ê, ó, ô, etc.)

  • ´ (accent aigu) puis voyelle → á, é, í, ó, ú
  • ^ (circonflexe) puis voyelle → â, ê, ô
  • Maj + ´ (accent grave) puis voyelle → à

Résumé : Avec AZERTY Global, vous pouvez écrire "coração" ou "São Paulo" sans difficulté !

Comment taper en italien (à, è, é, ì, ò, ù) ?

AZERTY Global est parfait pour l'italien car les voyelles accentuées sont très accessibles :

Les voyelles avec accent grave (à, è, ì, ò, ù)

  • à est directement sur le clavier (touche 0)
  • è est directement sur le clavier (touche 7)
  • Maj + ´ (accent grave) puis I → ì
  • Maj + ´ puis O → ò
  • AltGr + U → ù

L'accent aigu (é)

  • é est directement sur le clavier (touche 2)

Les majuscules accentuées

  • Verr. Maj puis la lettre accentuée → À, È, É
  • Ou ^ puis la lettre accentuée → majuscule automatique

Résumé : Avec AZERTY Global, vous pouvez écrire "perché" ou "città" naturellement !

Comment taper en langues nordiques (æ, ø, å) — Danois, Norvégien, Suédois ?

AZERTY Global supporte les caractères scandinaves :

Les lettres nordiques

  • AltGr + A → æ / AltGr + Maj + A → Æ
  • ´ (accent aigu) puis Q → ø / Maj + Q → Ø
  • ´ puis X → å / Maj + X → Å

Alternative pour å : AltGr + ) (touche morte anneau) puis A

Les umlauts (ä, ö)

  • Maj + ^ (tréma) puis A → ä
  • Maj + ^ puis O → ö

Résumé : Écrivez "København" ou "Malmö" facilement !

Comment taper en polonais (ą, ę, ł, ń, ś, ź, ż) ?

AZERTY Global inclut des raccourcis optimisés pour le polonais :

Via la touche circonflexe (^) — raccourcis rapides

  • ^ puis Q → ą
  • ^ puis D → ę
  • ^ puis L → ł
  • ^ puis M → ż

Via la touche accent aigu (´)

  • ´ puis N → ń
  • ´ puis S → ś
  • ´ puis C → ć
  • ´ puis Z → ź
  • ´ puis O → ó

Résumé : Écrivez "Dziękuję" ou "Łódź" sans problème !

Comment taper en tchèque/slovaque (č, š, ž, ř, ě) — langues à caron ?

AZERTY Global supporte le caron (háček) pour les langues slaves :

Via la touche circonflexe (^) — raccourcis

  • ^ puis F → č
  • ^ puis X → š

Via la touche morte caron

AltGr + ^ active la touche morte caron :

  • Caron + C → č
  • Caron + S → š
  • Caron + Z → ž
  • Caron + R → ř (tchèque)
  • Caron + E → ě
  • Caron + N → ň
  • Caron + D → ď
  • Caron + T → ť

Résumé : Écrivez "Čeština" ou "Žižkov" facilement !

Comment taper en roumain (ă, â, î, ș, ț) ?

AZERTY Global supporte les caractères roumains :

Via la touche circonflexe (^) — raccourcis

  • ^ puis R → ă
  • ^ puis T → ș
  • ^ puis A → â
  • ^ puis I → î

Le ț (t avec virgule)

  • AltGr + ç (touche morte virgule) puis T → ț

Résumé : Écrivez "București" ou "mulțumesc" sans difficulté !

Comment taper en hongrois (ő, ű, á, é, í, ó, ú) ?

AZERTY Global supporte les caractères hongrois, y compris le double aigu :

Le double aigu (ő, ű)

AltGr + é (touche 2) active la touche morte double aigu :

  • Double aigu + O → ő
  • Double aigu + U → ű

Les voyelles avec accent aigu

  • ´ (accent aigu) puis voyelle → á, é, í, ó, ú

Les umlauts (ö, ü)

  • Maj + ^ (tréma) puis voyelle → ö, ü

Résumé : Écrivez "Magyarország" ou "köszönöm" facilement !

Comment taper en turc (ş, ğ, ı, ö, ü, ç) ?

AZERTY Global supporte les caractères turcs :

Via la touche accent aigu (´) — raccourcis

  • ´ puis D → ş (s cédille)
  • ´ puis F → ğ (g brève)
  • ´ puis T → ı (i sans point)

Le İ (I avec point)

  • AltGr + & (touche morte point) puis Maj + I → İ

Les autres caractères

  • ç est directement sur le clavier
  • Maj + ^ puis O/U → ö, ü

Résumé : Écrivez "Türkiye" ou "teşekkürler" sans problème !

✒️ Ponctuation française

Comment taper la ponctuation française (« », —, espaces insécables, …) ?

AZERTY Global rend la typographie française professionnelle accessible :

Les guillemets français « »

  • AltGr + W → «
  • AltGr + X → »

W et X sont côte à côte sur la rangée du bas, facile à retenir !

Les tirets

  • AltGr + T → – (tiret demi-cadratin, pour les intervalles : 2020–2025)
  • AltGr + Maj + T → — (tiret cadratin, pour les dialogues)
  • AltGr + - → ‑ (tiret insécable)

Les espaces insécables

  • AltGr + Espace → espace insécable (avant : ; ! ?)
  • AltGr + Maj + Espace → espace fine insécable

L'espace insécable empêche les retours à la ligne indésirables avant la ponctuation.

Les points de suspension …

  • Maj + ! (touche morte ponctuation) puis . → …

L'apostrophe typographique '

  • Maj + ! (touche morte ponctuation) puis V → '

L'apostrophe typographique (') est plus élégante que l'apostrophe droite (').

Les guillemets anglais " "

  • AltGr + Maj + W → " (ouvrant)
  • AltGr + Maj + X → " (fermant)

Résumé : Avec AZERTY Global, vous pouvez écrire « Bonjour ! » — avec une typographie professionnelle…
