Télécharger AZERTY Global

Vous n’arrivez pas à taper É, Ç, À, œ, « », les caractères des langues africaines ou vous utilisez un clavier étranger ? AZERTY Global installe une disposition francophone améliorée sans repartir de zéro.

Majuscules accentuées faciles, point et arobase en accès direct, symboles dev mieux placés, et tous les accents pour écrire parfaitement en Europe et en Afrique francophone.

📖 Installation — Application Microsoft Store

1. Installez depuis le Store

Cliquez sur « Installer depuis le Microsoft Store » ci-dessus. L’application s’installe automatiquement, sans avertissement SmartScreen et sans droits administrateur.

2. Lancez l’application

Ouvrez AZERTY Global depuis le menu Démarrer. Au premier lancement, un guide de prise en main s’affiche :

3. L’icône AG dans la barre des tâches

L’icône AG apparaît dans la barre des tâches. Clic droit pour accéder au clavier virtuel, à la recherche de caractères et aux options.

4. Activez / désactivez à tout moment

Raccourci : Ctrl + Maj + Verr.Maj — ou clic droit sur l’icône AG → Désactiver.

💡 Fonctionnalités intégrées

  • Clavier virtuel (Ctrl+Maj+Q) : visualisez tous les caractères disponibles en temps réel.
  • Recherche de caractères (Ctrl+Maj+W) : trouvez comment taper n’importe quel caractère.
  • Lancement automatique : clic droit sur l’icône AG → Lancer au démarrage de Windows.
📖 Installation — Installateur EXE

1. Dézippez le fichier

Faites un clic droit sur AZERTY_Global_Windows.zip → « Extraire tout » ou ouvrez-le et extrayez le dossier.

2. Lancez l’installeur

Double-cliquez sur AZERTY_Global.exe. Vérifiez que ☑ « Add to language bar list » est coché, puis cliquez sur « Install ».

3. Vérifiez que c’est actif

Cliquez sur FRA dans la barre des tâches (en bas à droite). Si « AZERTY Global » apparaît → sélectionnez-le, c’est bon !

💡 Si AZERTY Global n’apparaît pas, ajoutez-le manuellement via Paramètres → Heure et langue → Langue et région → Français (⋯) → Options de langue → Ajouter un clavier.

💡 Si rien ne se passe en cliquant sur FRA, redémarrez l’ordinateur puis réessayez.

4. Changez de clavier

Deux méthodes :

  • Win + Espace pour basculer entre les claviers
  • Cliquez sur FRA dans la barre des tâches (en bas à droite) pour choisir

⭐ Garder AZERTY Global par défaut

Pour éviter que Windows rebascule sur votre ancien clavier après un redémarrage :

  • Option 1 (recommandé) : Supprimez les anciens claviers dont vous n’avez plus besoin (Paramètres → Langue → Français → Options → Claviers → Supprimer)
  • Option 2 : Si vous devez garder plusieurs claviers, définissez AZERTY Global par défaut : Paramètres → Saisie → Paramètres de clavier avancés → « Remplacer la méthode d’entrée par défaut » → choisissez AZERTY Global 2026

💡 Pour mettre AZERTY Global en tête de liste : enlevez puis rajoutez les autres claviers.

🗑️ Désinstallation

Application Microsoft Store :

Win + I → Applications → Applications installées → Cherchez « AZERTY Global » → Désinstaller.

Installateur EXE :

Win + I → Heure et langue → Langue et région → Français (⋯) → Options de langue → Sélectionnez AZERTY Global → Supprimer. Puis relancez l’installeur EXE pour désinstaller complètement.

✅ Les fichiers installés se retirent avec les étapes ci-dessus.

🛡️ Pourquoi j’ai un avertissement SmartScreen ?

Windows SmartScreen protège contre les logiciels non signés numériquement. AZERTY Global étant un projet indépendant, il n’est pas encore signé — mais il est 100% open source et vérifiable sur GitHub.

L’installateur EXE copie la disposition clavier dans C:\Windows\System32, le dossier système le plus sensible de Windows. C’est obligatoire — c’est exactement là que Windows attend de trouver les dispositions de claviers.

💡 Alternative sans SmartScreen : l’application Microsoft Store ne touche à aucun fichier système, ne nécessite aucun droit administrateur et ne déclenche aucun avertissement SmartScreen.

💡 Plus AZERTY Global sera téléchargé, plus la réputation SmartScreen du fichier pourra évoluer. Cet avertissement peut rester affiché selon Windows et le fichier téléchargé.

Pour passer l’avertissement :
  1. Cliquez sur « Informations complémentaires »
  2. Puis « Exécuter quand même »
🏪 Je n’ai pas accès à l’application sur le Microsoft Store

L’installation ne dépend pas du Store, mais l’option dépend de votre poste :

  • PC personnel sans Store (Windows 7/8, « N », LTSC, Server) : l’installateur EXE ci-dessus fonctionne (Windows 7 à 11). Il affiche un avertissement SmartScreen — voir la section juste au-dessus pour le passer.
  • Poste d’entreprise ou d’école (Store bloqué, sans droits admin) : l’installateur EXE est peu adapté (droits administrateur requis + SmartScreen). La solution propre — une version MSIX signée, déployable par le service informatique — arrive prochainement. En attendant, voir la page entreprises.

📖 Instructions d’installation

1. Ouvrez le Finder

Dans le menu « Aller » (en haut), maintenez la touche « Option » (Alt) enfoncée, puis cliquez sur « Bibliothèque » (Library) qui apparaît.

2. Accédez au dossier Keyboard Layouts

Ouvrez le dossier « Keyboard Layouts » (créez-le s’il n’existe pas).
Chemin complet : ~/Library/Keyboard Layouts/

3. Copiez le fichier .keylayout

Copiez le fichier AZERTY Global.keylayout dans ce dossier.

💡 Vous pouvez aussi l’installer pour tous les utilisateurs dans /Library/Keyboard Layouts/ (nécessite un mot de passe administrateur).

4. Redémarrez votre session

Redémarrez votre session ou votre Mac (obligatoire pour que macOS détecte le fichier).

⭐ Activer le clavier

Une fois la session redémarrée :

  1. Ouvrez « Réglages Système » (ou « Préférences Système »)
  2. Allez dans « Clavier » → « Entrée de texte » → « Modifier... »
  3. Cliquez sur le « + » en bas à gauche pour ajouter un clavier
  4. Dans la liste, cherchez « Autres » (tout en bas) ou « Français »
  5. Sélectionnez « AZERTY Global » et cliquez sur « Ajouter »

🎯 Utilisation

Cliquez sur l’icône de clavier dans la barre des menus (en haut à droite, près de l’heure) et sélectionnez « AZERTY Global ».

💡 Raccourci : Vous pouvez basculer entre les claviers avec Ctrl + Espace (ou Cmd + Espace selon vos réglages).

🗑️ Désinstallation

Retirer le clavier :

  1. Ouvrez « Réglages Système » → « Clavier » → « Entrée de texte »
  2. Sélectionnez « AZERTY Global » et cliquez sur « - »
  3. Supprimez le fichier AZERTY Global.keylayout de ~/Library/Keyboard Layouts/
  4. Redémarrez votre session

✅ Les fichiers installés se retirent avec les étapes ci-dessus.

📖 Instructions d’installation

1. Dézippez et ouvrez un terminal

Extrayez le fichier ZIP, puis ouvrez un terminal dans le dossier extrait.

2. Lancez le script d’installation

chmod +x install.sh
sudo ./install.sh

3. Redémarrez votre session

Déconnectez-vous puis reconnectez-vous (ou redémarrez).

4. Activez le clavier

GNOME / Ubuntu : Paramètres → Clavier → Sources de saisie → + → Cherchez « French (AZERTY Global) »

KDE / Plasma : Configuration système → Clavier → Dispositions → Ajouter → « French (AZERTY Global) »

🗑️ Désinstallation

Retirer le clavier :

  1. Retirez « AZERTY Global » de vos sources de saisie dans les paramètres
  2. Supprimez le fichier : sudo rm /usr/share/X11/xkb/symbols/azerty_global
  3. Restaurez votre ancien ~/.XCompose si besoin (un backup a été créé)
  4. Redémarrez votre session
