Ce qui change avec AZERTY Global

99% des habitudes préservées. Les quelques changements sont tous bénéfiques.

Problème AZERTY Windows AZERTY Global
Majuscules accentuées ❌ Alt+144 ou copier-coller ✅ Verr. Maj + é = É
Le point ❌ Maj + ; ✅ Accès direct
L'arobase @ ❌ AltGr + 0 ✅ Accès direct (touche dédiée)
Symboles dev { } [ ] | \ ❌ Difficiles d'accès ✅ AltGr + rangée de repos
Guillemets « », tirets —, œ, æ ❌ Absents ou codes Alt ✅ Accessibles facilement
Langues étrangères (ñ, ß, ø…) ❌ Codes Alt obscurs ✅ Touches mortes intuitives

Une réponse concrète à un problème reconnu

En 2019, l'AFNOR a publié la norme NF Z71-300, reconnaissant officiellement les lacunes du clavier AZERTY français. Mais cette norme est restée largement méconnue : peu d’établissements la connaissent, et ceux qui la connaissent ne l’appliquent pas. La raison ? Trop de symboles changent de place, ce qui rend en pratique l'achat de claviers physiques neufs quasi incontournable pour les entreprises — une solution peu réaliste pour les budgets scolaires.

AZERTY Global résout le même problème sans changer de matériel. Porté par l'Association pour la Modernisation du Clavier Français (AMCF), association loi 1901, le projet est publié sous licence EUPL 1.2 (open source européenne). Code source disponible sur GitHub.

Questions fréquentes

Doit-on changer les claviers physiques ?

Non. AZERTY Global est une couche logicielle. Vos claviers actuels suffisent.

Faut-il coller des stickers sur les touches ?

Non. Les touches principales ne bougent pas. Une carte mémo est disponible pour les élèves.

Est-ce compatible avec les logiciels scolaires ?

Oui. LibreOffice, suite Google, ENT, navigateurs — AZERTY Global est reconnu comme un clavier standard par le système.

Peut-on revenir à l'AZERTY classique ?

Oui. Ctrl+Maj+Verr. Maj désactive l’application instantanément. Sur macOS/Linux, changement dans les paramètres de langue.

Le logiciel collecte-t-il des données ?

Non. Outil 100% local, conforme RGPD. Aucune donnée transmise.

À partir de quel âge peut-on l’utiliser ?

Dès que l’élève utilise un clavier physique (généralement CE2/CM1). Le mode leçons est adapté à tous les niveaux.
