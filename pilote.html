Questions pratiques

Faut-il changer de clavier physique ?

Non. Le pilote se fait sur les claviers existants : AZERTY Global est une disposition logicielle.

Qui installe la disposition ?

Le test se prépare avec une personne référente côté structure. L’installation dépend ensuite de vos règles internes et du système utilisé.

Peut-on revenir en arrière ?

Oui. Le pilote est réversible : il est toujours possible de revenir à la disposition habituelle.

Quelles données sont collectées ?

Le formulaire transmet uniquement les informations que vous saisissez pour organiser le pilote. L’usage du clavier reste local sur le poste.

Combien de temps dure le pilote ?

Le format recommandé est de 2 semaines, avec une prolongation possible à 4 semaines si le test mérite d’être poursuivi.

Comment gérer les élèves mineurs ?

Les retours peuvent être agrégés par la personne référente de l’établissement. La page ne demande pas de collecte nominative auprès de personnes mineures.
