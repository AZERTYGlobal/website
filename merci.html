🛡️ Un avertissement Windows SmartScreen apparaît ?

Si vous avez installé via le Microsoft Store, vous n’avez pas cet avertissement.
Pour l’installateur EXE, c’est normal : les dispositions de clavier s’installent dans le dossier système de Windows. AZERTY Global est 100% open source et son code est vérifiable sur GitHub.

Pour continuer : cliquez « Informations complémentaires » puis « Exécuter quand même ».

⚠️ Conflits potentiels avec certains logiciels

Sur Windows, AltGr équivaut à Ctrl+Alt. Certains logiciels (IDE JetBrains) peuvent intercepter ces combinaisons et bloquer l’accès aux caractères comme { } [ ]. Ce problème se résout facilement en remappant le raccourci concerné dans les paramètres du logiciel. La plupart des applications (Word, Excel, navigateurs) fonctionnent parfaitement.

📦 Prochaines étapes

1️⃣

Installez AZERTY Global

Application Microsoft Store : ouvrez l’app depuis le menu Démarrer. L’icône AG apparaît dans la barre des tâches — c’est prêt !

  • Ctrl+Maj+Verr.Maj pour activer/désactiver
  • Ctrl+Maj+Q pour le clavier virtuel
  • Ctrl+Maj+W pour la recherche de caractères

Installateur EXE : lancez le fichier téléchargé, vérifiez que ☑ « Add to language bar list » est coché, puis cliquez sur « Install ». Redémarrez votre session. Sélectionnez AZERTY Global dans la barre des tâches (Win+Espace). Un avertissement SmartScreen peut apparaître → en savoir plus.

  1. Ouvrez le Finder. Dans le menu « Aller », maintenez la touche Option (Alt) enfoncée et cliquez sur Bibliothèque.
  2. Ouvrez le dossier Keyboard Layouts (créez-le s’il n’existe pas). Chemin : ~/Library/Keyboard Layouts/
  3. Copiez le fichier AZERTY Global (Beta).keylayout dans ce dossier.
  4. Redémarrez votre session (obligatoire pour que macOS détecte le fichier).
  5. Ouvrez Réglages Système → Clavier → Entrée de texte → Modifier…
  6. Cliquez sur +, cherchez « Autres » ou « Français », sélectionnez AZERTY Global (Beta) → Ajouter.

💡 Basculez entre vos claviers avec Ctrl+Espace ou via l’icône clavier dans la barre des menus.

  1. Ouvrez un terminal dans le dossier téléchargé.
  2. Rendez le script exécutable et lancez-le :
    chmod +x install.sh && sudo ./install.sh
  3. Redémarrez votre session (déconnexion/reconnexion).
  4. GNOME/Ubuntu : Paramètres → Clavier → Sources de saisie (+) → French (AZERTY Global Beta).
  5. KDE/Plasma : Configuration du système → Périphériques d’entrée → Clavier → Dispositions → French (AZERTY Global Beta).
