Le détail des raccourcis

La majorité des symboles ne bougent pas. Ceux qui changent gagnent une position plus logique.

La ceinture de code

Les symboles qui structurent le code, regroupés sur la rangée de repos et atteignables sans bouger les mains — avec AltGr :

  • AltGr + D = { · AltGr + F = }
  • AltGr + J = [ · AltGr + K = ]
  • AltGr + G = \ · AltGr + H = |

Les symboles optimisés

Les caractères les plus utiles au code, ramenés à une position naturelle :

  • @ en accès direct (touche en haut à gauche)
  • # sur Maj + @, ou AltGr + : pour les devs
  • ^ sur AltGr + I — le caret direct, sans touche morte
  • ` sur AltGr + L
  • ~ sur AltGr + N
  • < > en accès direct sur la touche dédiée (ou AltGr + , / AltGr + ; sur clavier ANSI)
