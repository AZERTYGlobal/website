C'est quoi AZERTY Global ?

Le clavier français corrigé, pas réinventé.

⌨️ En 30 secondes

AZERTY Global est une disposition de clavier gratuite et open source qui corrige les défauts historiques de l'AZERTY Windows. Vous pouvez enfin taper É, È, À, Ç directement avec Verrouillage Majuscule, utiliser les guillemets français « », et accéder aux symboles de programmation { } [ ] facilement. Le tout sans bouleverser vos habitudes : seulement 5 changements bénéfiques et essentiels.

⚠️ À ne pas confondre avec l'AZERTY AFNOR

AZERTY Global est un projet indépendant, différent de la norme AFNOR (NF Z71-300 de 2019). Les deux dispositions partagent certains objectifs mais diffèrent dans leur approche. AZERTY Global privilégie une transition douce depuis l'AZERTY Windows traditionnel.

🔄 Ce qui change concrètement

🔤

Majuscules accentuées

Verr. Maj + é → É. Fini les Alt+144 ou le copier-coller.

✨

Typographie française

Guillemets « », tirets —, œ, æ, espaces insécables accessibles.

💻

Symboles dev

{ } [ ] | \ sur la rangée de repos avec AltGr. Idéal pour coder.

🌍

Langues étrangères

Latin, cyrillique, grec via touches mortes. Espagnol ñ, allemand ß, danois ø...

📊

Symboles

Scientifiques (± × ÷ ≈), monétaires (€ £ ¥), et divers (© ® ™ …).

❓ Questions rapides

Est-ce gratuit ?

Oui, 100% gratuit et open source. Le code est disponible sur GitHub sous licence EUPL 1.2.

Dois-je acheter un nouveau clavier ?

Non. AZERTY Global est une solution purement logicielle. Elle fonctionne avec votre clavier actuel.

C'est compatible avec les jeux (ZQSD) ?

Oui. Les touches physiques Z, Q, S, D ne bougent pas. Vos jeux fonctionnent exactement comme avant.

Disponible sur Mac et Linux ?

Oui ! AZERTY Global est disponible sur Windows (7, 8, 10, 11), macOS (10.12+) et Linux (Ubuntu, Debian, Fedora, Arch...).

Combien de temps pour s'adapter ?

Quelques minutes à quelques heures. Seulement 5 changements bénéfiques et essentiels, moins de 1% de vos frappes seront différentes.

Qui a créé AZERTY Global ?

Antoine Olivier, particulier français, depuis 2017. Projet indépendant et bénévole.

🏗️ Les fondations d'AZERTY Global

AZERTY Global n'est pas né de nulle part. Sa conception s'appuie sur plusieurs sources :

  • Enquête publique AFNOR de juin 2017 — Les retours de milliers de Français sur les problèmes du clavier AZERTY ont orienté les priorités de développement.
  • Étude des dispositions étrangères — Inspiration des bonnes pratiques internationales, comme les touches mortes sur la rangée numérique (disposition hongroise) ou le Verrouillage Majuscule intelligent (disposition suisse).
  • Intuition et pratique quotidienne — 9 ans d'utilisation et d'amélioration continue de la disposition par son créateur.
  • Retours de la communauté — Les suggestions des bêta-testeurs ont façonné les versions successives.
