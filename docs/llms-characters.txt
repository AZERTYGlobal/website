# AZERTY Global — Index des caractères

Ce fichier contient l'emplacement de tous les caractères sur le clavier AZERTY Global.
Format : caractère → raccourci clavier

## Légende des touches

- Base : touche seule
- Shift (Maj) : Maj + touche
- AltGr : AltGr + touche (touche Alt de droite)
- Caps (Verr. Maj) : Verr. Maj activé + touche
- Touche morte : séquence de 2 touches

## Caractères courants français

### Majuscules accentuées (plusieurs méthodes)

É (e accent aigu majuscule)
  → Verr. Maj + é (touche 2) — recommandé
  → ^ puis é (circonflexe puis e accent aigu)
  → ´ puis Shift + E (accent aigu puis E majuscule)

È (e accent grave majuscule)
  → Verr. Maj + è (touche 7) — recommandé
  → ^ puis è (circonflexe puis e accent grave)
  → ` puis Shift + E (accent grave puis E majuscule)

À (a accent grave majuscule)
  → Verr. Maj + à (touche 0) — recommandé
  → ^ puis à (circonflexe puis a accent grave)
  → ` puis Shift + A (accent grave puis A majuscule)

Ç (c cédille majuscule)
  → Verr. Maj + ç (touche 9) — recommandé
  → ^ puis ç (circonflexe puis c cédille)
  → ¸ puis Shift + C (cédille puis C majuscule)

ù (u accent grave minuscule)
  → AltGr + U — recommandé
  → ´ puis V (accent aigu puis v) — rapide
  → ` puis U (accent grave puis u)

Ù (u accent grave majuscule)
  → Shift + AltGr + U — recommandé
  → ´ puis Shift + V (accent aigu puis V majuscule) — rapide
  → Verr. Maj + AltGr + U
  → ` puis Shift + U (accent grave puis U majuscule)

### Typographie française
« → AltGr + W (guillemet ouvrant)
» → AltGr + X (guillemet fermant)
œ → AltGr + O
æ → AltGr + A
— → AltGr + - (tiret cadratin, touche 6)
– → AltGr + Shift + - (tiret demi-cadratin)
  → AltGr + Espace (espace insécable)
… → AltGr + . (points de suspension)
' → AltGr + ' (apostrophe typographique)

## Symboles de programmation

### Accès direct ou AltGr
@ → touche ² (à gauche du 1) — accès direct
# → Shift + @ (touche ²)
% → Shift + ) (touche - en haut) — nouvel emplacement
{ → AltGr + D (accolade ouvrante)
} → AltGr + F (accolade fermante)
[ → AltGr + J (crochet ouvrant)
] → AltGr + K (crochet fermant)
| → AltGr + H (barre verticale, pipe)
\ → AltGr + G (backslash)
€ → AltGr + E

### Tilde et backtick (touche N)
~ → AltGr + N (tilde vif, caractère direct, pas une touche morte)
` → Shift + AltGr + N (backtick vif, caractère direct)

Note : Ces caractères s'affichent immédiatement, contrairement aux touches mortes.
La touche morte tilde est différente (pour faire ñ, ã, etc.).

### Symboles mathématiques
± → touche morte scientifique + Espace
× → touche morte scientifique + X
÷ → touche morte scientifique + :
≠ → touche morte scientifique + =
≤ → touche morte scientifique + <
≥ → touche morte scientifique + >
∞ → touche morte scientifique + 8
√ → touche morte scientifique + V

## Lettres et symboles étrangers

### Espagnol
ñ → touche morte tilde (~) puis N
Ñ → touche morte tilde puis Shift + N
¿ → AltGr + Shift + ? (point d'interrogation inversé)
¡ → AltGr + ! (point d'exclamation inversé)

### Allemand
ß → AltGr + S (eszett)
ü → touche morte tréma (¨) puis U
ö → touche morte tréma puis O
ä → touche morte tréma puis A

### Portugais/Langues nordiques
ã → touche morte tilde puis A
õ → touche morte tilde puis O
ø → touche morte barre oblique (/) puis O
å → touche morte rond (°) puis A

### Autres accents
á → touche morte accent aigu (´) puis A
à → touche morte accent grave (`) puis A
â → touche morte accent circonflexe (^) puis A
ä → touche morte tréma (¨) puis A
ã → touche morte tilde (~) puis A
ā → touche morte macron (¯) puis A
ą → touche morte ogonek (˛) puis A
ǎ → touche morte caron (ˇ) puis A

## Symboles monétaires
€ → AltGr + E
£ → Shift + $ (touche ])
$ → touche ] (accès direct)
¥ → touche morte monnaies + Y
¢ → touche morte monnaies + C

## Accès aux touches mortes

Touche morte accent aigu (´) → AltGr + , (virgule)
Touche morte accent grave (`) → AltGr + Shift + N
Touche morte tréma (¨) → Shift + ^ (touche ^)
Touche morte circonflexe (^) → ^ (touche ^, accès direct)
Touche morte tilde (~) → AltGr + N puis relâcher, puis lettre
Touche morte cédille (¸) → AltGr + C
Touche morte macron (¯) → AltGr + Shift + ,
Touche morte caron (ˇ) → AltGr + Shift + ^ 
Touche morte ogonek (˛) → AltGr + Shift + C
Touche morte rond (°) → AltGr + -

## Remarques

- ZQSD : les touches de jeu ne bougent pas
- 5 changements principaux par rapport à l'AZERTY Windows :
  1. @ en accès direct (remplace ²)
  2. < > déplacés
  3. Verr. Maj intelligent (É È À Ç)
  4. Symboles dev sur rangée de repos
  5. Typographie française accessible
