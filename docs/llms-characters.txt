# AZERTY Global — Index des caractères

Ce fichier contient l'emplacement de tous les caractères sur le clavier AZERTY Global.
Format : caractère → raccourci clavier

## Légende des touches

- Base : touche seule
- Shift (Maj) : Maj + touche
- AltGr : AltGr + touche (touche Alt de droite)
- Caps (Verr. Maj) : Verr. Maj activé + touche
- Touche morte : séquence de 2 touches

## Caractères courants français

### Majuscules accentuées (plusieurs méthodes)

É (e accent aigu majuscule)
  → Verr. Maj + é (touche 2) — recommandé
  → ^ puis é (circonflexe puis e accent aigu)
  → ´ puis Shift + E (accent aigu puis E majuscule)

È (e accent grave majuscule)
  → Verr. Maj + è (touche 7) — recommandé
  → ^ puis è (circonflexe puis e accent grave)
  → ` puis Shift + E (accent grave puis E majuscule)

À (a accent grave majuscule)
  → Verr. Maj + à (touche 0) — recommandé
  → ^ puis à (circonflexe puis a accent grave)
  → ` puis Shift + A (accent grave puis A majuscule)

Ç (c cédille majuscule)
  → Verr. Maj + ç (touche 9) — recommandé
  → ^ puis ç (circonflexe puis c cédille)
  → ¸ puis Shift + C (cédille puis C majuscule)

ù (u accent grave minuscule)
  → AltGr + U — recommandé
  → ´ puis V (accent aigu puis v) — rapide
  → ` puis U (accent grave puis u)

Ù (u accent grave majuscule)
  → Shift + AltGr + U — recommandé
  → ´ puis Shift + V (accent aigu puis V majuscule) — rapide
  → Verr. Maj + AltGr + U
  → ` puis Shift + U (accent grave puis U majuscule)

### Typographie française
« → AltGr + W (guillemet ouvrant)
» → AltGr + X (guillemet fermant)
œ → AltGr + O (Œ majuscule : Shift + AltGr + O)
æ → AltGr + A (Æ majuscule : Shift + AltGr + A)
— → AltGr + Shift + T (tiret cadratin, touche T)
– → AltGr + T (tiret demi-cadratin)
  → AltGr + Espace (espace insécable)
… → touche morte ponctuation (Shift + !) puis . (points de suspension)
' → AltGr + ' (apostrophe typographique, touche 4)
' → Shift + AltGr + ' (apostrophe ouvrante, touche 4)
" → Shift + AltGr + W (guillemet anglais ouvrant)
" → Shift + AltGr + X (guillemet anglais fermant)
° → AltGr + ) (symbole degré, touche E11)
· → Shift + AltGr + . (point médian, écriture inclusive)
  → Shift + AltGr + Espace (espace fine insécable, avant ; : ! ?)

### Accents via touches mortes (les 5 touches mortes de base)

Une touche morte ne produit rien seule : on appuie dessus, puis on appuie sur la lettre à accentuer.

Accent circonflexe (^) → touche ^ (D11, accès direct)
  Appuyer ^ puis la lettre : a→â, e→ê, i→î, o→ô, u→û

Tréma (¨) → Shift + ^ (D11)
  Appuyer Shift+^ puis la lettre : e→ë, i→ï, u→ü, o→ö, a→ä

Accent aigu (´) → touche ù (C11, accès direct)
  Appuyer ´ puis la lettre : e→é, a→á, i→í, o→ó, u→ú

Accent grave (`) → Shift + touche ù (C11)
  Appuyer ` puis la lettre : e→è, a→à, u→ù, i→ì, o→ò

Tilde (~) → AltGr + touche ù (C11)
  Appuyer la touche morte tilde puis la lettre : n→ñ, a→ã, o→õ

Cédille (¸) → AltGr + ç (E09, touche 9)
  Appuyer ¸ puis la lettre : c→ç, s→ş, t→ţ

Note : la touche morte accent aigu (´) remplace le ù de l'AZERTY Windows sur cette touche.
Le ù est désormais sur AltGr + U.

### Exposants et indices (via touches mortes)

Exposants → touche morte ^ puis chiffre : 0→⁰, 1→¹, 2→², 3→³, 4→⁴... 9→⁹
  Aussi : +→⁺, -→⁻, =→⁼, (→⁽, )→⁾
  Exemple : x² = x puis ^ puis 2

Indices → touche morte ´ puis chiffre : 0→₀, 1→₁, 2→₂, 3→₃, 4→₄... 9→₉
  Aussi : +→₊, -→₋, =→₌, (→₍, )→₎
  Exemple : H₂O = H puis ´ puis 2 puis O

## Symboles de programmation

### Accès direct ou AltGr
@ → touche ² (à gauche du 1) — accès direct
# → Shift + @ (touche ²)
% → Shift + ) (touche - en haut) — nouvel emplacement
{ → AltGr + D (accolade ouvrante)
} → AltGr + F (accolade fermante)
[ → AltGr + J (crochet ouvrant)
] → AltGr + K (crochet fermant)
| → AltGr + H (barre verticale, pipe)
\ → AltGr + G (backslash)
€ → AltGr + E

### Tilde et backtick (touche N)
~ → AltGr + N (tilde vif, caractère direct, pas une touche morte)
` → Shift + AltGr + N (backtick vif, caractère direct)

Note : Ces caractères s'affichent immédiatement, contrairement aux touches mortes.
La touche morte tilde est différente (pour faire ñ, ã, etc.).

### Symboles mathématiques
≤ → AltGr + < (touche B00, accès direct)
≥ → Shift + AltGr + > (touche B00, accès direct)
± → touche morte scientifique + Espace
× → touche morte scientifique + X
÷ → touche morte scientifique + /
≠ → touche morte scientifique + =
∞ → touche morte scientifique + 8
√ → touche morte scientifique + v (minuscule ; V majuscule donne ∛)

## Lettres et symboles étrangers

### Espagnol
ñ → touche morte tilde (~) puis N
Ñ → touche morte tilde puis Shift + N
¿ → AltGr + Shift + ? (point d'interrogation inversé)
¡ → AltGr + ! (point d'exclamation inversé)

### Allemand
ß → AltGr + B (eszett)
ü → touche morte tréma (¨) puis U
ö → touche morte tréma puis O
ä → touche morte tréma puis A

### Portugais/Langues nordiques
ã → touche morte tilde puis A
õ → touche morte tilde puis O
ø → touche morte barre oblique (/) puis O
å → touche morte rond (°) puis A

### Autres diacritiques (touches mortes avancées)
ā → touche morte macron (¯) puis la lettre (AltGr + _, touche 8)
ą → touche morte ogonek (˛) puis la lettre (Shift + AltGr + ^, touche D11)
ǎ → touche morte caron (ˇ) puis la lettre (AltGr + ^, touche D11)
đ → touche morte barre horizontale puis D (Shift + AltGr + è, touche 7)

## Symboles monétaires
€ → AltGr + E
£ → Shift + $ (touche ])
$ → touche ] (accès direct)
¥ → AltGr + $ (touche $, D12)
¢ → touche morte monnaies + C

## Accès aux touches mortes

Touche morte accent aigu (´) → touche ù (C11, accès direct)
Touche morte accent grave (`) → Shift + touche ù (C11)
Touche morte tilde (~) → AltGr + touche ù (C11)
Touche morte tréma (¨) → Shift + ^ (D11)
Touche morte circonflexe (^) → ^ (D11, accès direct)
Touche morte caron (ˇ) → AltGr + ^ (D11)
Touche morte ogonek (˛) → Shift + AltGr + ^ (D11)
Touche morte cédille (¸) → AltGr + ç (E09, touche 9)
Touche morte macron (¯) → AltGr + _ (E08, touche 8)
Touche morte rond (°) → Shift + AltGr + ) (E11, la touche avec ) et %)
Note : AltGr + ) seul donne le symbole ° (degré) directement, pas la touche morte.
Touche morte barre oblique (/) → AltGr + è (E07, touche 7)
Touche morte scientifique → AltGr + = (E12)
Touche morte grec → Shift + * (C12)
Touche morte cyrillique → AltGr + * (C12)
Touche morte symboles divers → AltGr + @ (E00)
Touche morte ponctuation → Shift + ! (B10)
Touche morte monnaies → Shift + AltGr + $ (D12)

Note : ~ sur AltGr + N et ` sur Shift + AltGr + N sont des caractères VIFS (directs),
pas des touches mortes. La touche morte tilde est sur AltGr + touche ù.

## Remarques

- ZQSD : les touches de jeu ne bougent pas
- 5 changements par rapport à l'AZERTY Windows :
  1. Verrouillage Majuscule intelligent (É È À Ç en un appui)
  2. Point en accès direct (point et point-virgule échangés)
  3. @ et # sur la touche ² (@ accessible directement)
  4. Symboles de programmation sur la rangée de repos ({ } [ ] | \ avec AltGr)
  5. Accents internationaux sur la touche ù (aigu ´ et grave ` remplacent le ù et le % qui sont déplacés)
