Les 5 changements d’AZERTY Global

Tout ce qu’il faut savoir en 2 minutes.

1. Verrouillage majuscule intelligent

Avec Verr. Maj. activé, les lettres (A-Z) et les lettres accentuées deviennent majuscules : é → É, è → È, à → À, ç → Ç.

La ponctuation reste de la ponctuation, les chiffres restent des chiffres.

→ En savoir plus : É majuscule, È majuscule, À majuscule, Ç majuscule

2. Le point en accès direct

Fini la touche Majuscule ! Le point . est maintenant en accès direct, comme partout ailleurs dans le monde. Le point-virgule passe en Maj + ..

💡 Le point est bien plus fréquent que le point-virgule — il mérite un accès direct.

3. Arobase @ et hashtag # en haut à gauche

L’arobase @ et le hashtag # sont maintenant en accès direct et avec Maj sur l’ancienne touche ² à gauche de la touche 1 — comme sur l’AZERTY de macOS. Le hashtag reste aussi disponible avec AltGr + : pour les développeurs. Taper des adresses mails n’a jamais été aussi facile !

4. Symboles de programmation accessibles

Fini les extensions de main : les symboles dev sont regroupés sur la rangée de repos avec AltGr, encore plus accessibles que sur le QWERTY américain :

{ } sous le majeur et l’index main gauche (D, F)
\ | sur les deux touches du milieu (G, H)
[ ] sous l’index et le majeur main droite (J, K)

Le tilde vif ~ est sur AltGr + N, le backtick vif ` sur AltGr + L, et le circonflexe vif ^ sur AltGr + I.

→ Voir la page développeurs pour tous les détails

5. Accents internationaux sur la touche ù

Le ù et le % ont été déplacés pour laisser place à deux accents :

• Accent aigu ´ — pour taper á, é, í, ó, ú (espagnol, hongrois…)
• Accent grave ` — pour taper à, è, ì, ò, ù (italien, français…)

💡 Le ù reste accessible via AltGr + U ou ´ puis V. Le % est sur Maj + ).
