Pourquoi AZERTY Global ?

Une disposition de clavier française pensée pour écrire correctement, sans tout réapprendre.

🔤

Majuscules accentuées

Tapez directement É, È, À, Ç avec Verrouillage Majuscule. Fini le copier-coller ou les codes ASCII.

✨

Typographie parfaite

Guillemets français « », tirets cadratins, espaces insécables — tout est accessible facilement.

💻

Ami des développeurs

Les symboles { } [ ] | \ sont sur la rangée de repos avec AltGr.
