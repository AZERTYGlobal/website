Manuel d'Utilisation

1. Pourquoi AZERTY Global ?

L'AZERTY standard a été conçu pour les machines à écrire, pas pour l'informatique moderne. AZERTY Global corrige ces défauts historiques :

  • Le Point (.) en accès direct : Inversion des touches ; et . pour un accès immédiat.
  • Majuscules Faciles : Verr. Maj active les majuscules accentuées (É, È, Ç, À).
  • Symboles Tech : @, #, {, }, [, ] sont placés logiquement pour les développeurs.

2. Installation

Windows : Installez via Setup.exe. Redémarrez. Sélectionnez "Français (AZERTY Global)".
Linux : Copiez les fichiers azerty-global et XCompose fournis.
macOS : Copiez AZERTY Global.keylayout dans /Library/Keyboard Layouts/.
