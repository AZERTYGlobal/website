Compatibilité logicielle

AZERTY Global fonctionne avec la grande majorité des logiciels. Voici les rares cas où un réglage peut être nécessaire.

Pourquoi certains conflits existent ?

Sur Windows, AltGr envoie Ctrl+Alt au système. Les applications modernes (Word, Excel, navigateurs) distinguent correctement les deux. Quelques logiciels basés sur Java ou Electron les confondent, ce qui peut bloquer certains caractères AltGr. La solution est toujours la même : remapper le raccourci en conflit.

Applications

Cliquez sur une application pour voir les conflits et la solution.

VS Code — 2 conflits

Résolution : Ctrl+K Ctrl+S pour ouvrir les raccourcis, puis désactiver les entrées en conflit. Ou ajouter dans keybindings.json une entrée sans commande.

  • Ctrl+Alt+[ bloque [ (AltGr+J) — Jump to Matching Bracket
  • Ctrl+Alt+] bloque ] (AltGr+K) — Jump to Matching Bracket
JetBrains (IntelliJ, PyCharm, WebStorm) — 7 conflits

Résolution : Settings > Keymap, clic droit sur l'action > Remove Shortcut.

Astuce : dans le registre IntelliJ (Ctrl+Maj+A > Registry), désactiver actionSystem.force.alt.gr pour résoudre tous les conflits d'un coup.

  • Ctrl+Alt+N bloque ~ (AltGr+N) — Inline Refactor
  • Ctrl+Alt+O bloque œ (AltGr+O) — Optimiser les imports
  • Ctrl+Alt+T bloque – (AltGr+T) — Surround With
  • Ctrl+Alt+B bloque ß (AltGr+B) — Aller à l'implémentation
  • Ctrl+Alt+H bloque | (AltGr+H) — Hiérarchie d'appels
  • Ctrl+Alt+J bloque [ (AltGr+J) — Surround with Live Template
  • Ctrl+Alt+F bloque } (AltGr+F) — Extraire champ
Notepad++ — 4 conflits

Résolution : Settings > Shortcut Mapper, onglet Main menu, double-cliquer sur l'action pour la modifier.

  • Ctrl+Alt+U bloque ù (AltGr+U) — Sentence case
  • Ctrl+Alt+B bloque ß (AltGr+B) — Sélectionner entre accolades
  • Ctrl+Alt+Espace bloque l'espace insécable (AltGr+Espace) — Path completion
  • Ctrl+Alt+F bloque } (AltGr+F) — Replier le code
Adobe Photoshop — 3 conflits

Résolution : Edit > Keyboard Shortcuts (Alt+Maj+Ctrl+K).

  • Ctrl+Alt+A bloque æ (AltGr+A) — Sélectionner tous les calques
  • Ctrl+Alt+W bloque « (AltGr+W) — Fermer tout
  • Ctrl+Alt+G bloque \ (AltGr+G) — Masque d'écrêtage
Figma — 1 conflit non remappable
  • Ctrl+Alt+G bloque \ (AltGr+G) — Frame Selection

Figma ne permet pas de remapper ses raccourcis. Utilisez le menu contextuel (clic droit) ou basculez temporairement vers l'AZERTY traditionnel avec Win+Espace.

Discord — 1 conflit

Résolution : Paramètres > Raccourcis clavier, désactiver le raccourci concerné.

  • Ctrl+Alt+A bloque æ (AltGr+A) — Retourner au canal vocal actif
Slack — 1 conflit non remappable
  • Ctrl+Alt+T bloque – (AltGr+T) — Fil de commentaires (Canvas)

Les raccourcis Canvas de Slack ne sont pas remappables. Basculez temporairement avec Win+Espace.

Jeux vidéo

La touche ² (console de jeux)

AZERTY Global remplace ² par @ sur la touche à gauche du 1. L'impact dépend de la méthode de détection du jeu :

  • Par scancode (position physique) : la console fonctionne normalement.
    ✅ Skyrim, Fallout 4, Counter-Strike 2, CS:GO.
  • Par caractère (virtual key) : la console peut ne pas s'ouvrir.
    ⚠️ Jeux Source Engine anciens (Garry's Mod, TF2, Portal), certains jeux Unreal Engine et Unity.

Solutions : rebinder la console dans le fichier INI du jeu, ou basculer vers l'AZERTY traditionnel avec Win+Espace.

Le Verr. Maj intelligent

Le Verr. Maj d'AZERTY Global n'affecte que les lettres. Les jeux détectent l'appui sur la touche, pas l'état du Verr. Maj : aucun impact connu, y compris dans les MMO et FPS qui rebindent Verr. Maj (WoW, Valorant).

Anti-triche

Aucune incompatibilité documentée avec EasyAntiCheat, BattlEye ou Vanguard. AZERTY Global utilise un pilote clavier standard (MSKLC), sans hook système.
